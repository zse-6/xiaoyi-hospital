# 晓医智能导诊小程序 
 
## 项目简介 
基于微信小程序的智能医疗分诊导诊系统，对应毕业论文《智能医疗分诊导诊微信小程序的设计与实现》。 
 
## 功能模块 
1. 智能分诊（规则引擎+决策树） 
2. 多模态输入（语音、文本、人体图示） 
3. 院内导航（图文+实景指引） 
4. 就诊流程管理（任务清单+提醒） 
 
## 项目结构 
- cloudfunctions/ - 云函数 
- miniprogram/ - 小程序前端 
- data-tools/ - 数据处理工具（论文重要部分） 
- docs/ - 文档记录（论文支持） 
 
## 快速开始 
1. 导入微信开发者工具 
2. 配置云环境ID 
3. 部署云函数 
4. 运行小程序
